# 排序算法整理

### 快排
```
function quick_sort(arr)
{
    if(arr.length<=1) return arr;
    let pivot=arr[0],
        left=[],
        right=[];
    
    for(let i=1; i<arr.length ; i++){
        if(arr[i]<pivot){
            left.push(arr[i])
        }else{
            right.push(arr[i]);
        }
    }

    return quick_sort(left).concat([pivot],quick_sort(right));
    
}

quick_sort([4,6,2,3,2]);
```
